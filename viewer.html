Summary

Reworked the STL file in Meshmixer. The original file posted by ThatOneGuyWhoDoesThings had a lot messed up edges. I cleaned these up and thickened up some parts like DK's cowlick and tie so they can print better. It's far from perfect (this is my first experience using Meshmixer to a significant degree), but I think its an improvement.

Print Settings

Printer brand: Creality

Printer: Ender 3 Pro

Rafts: No

Supports: Yes

Resolution: 0.4mm nozzle, 0.2mm layer height

Infill: 20%

Notes:

I used PrussaSlicer with Filament Friday's PrussaSlicer configuration for Ender 3 printers. I printed with supports and Donkey Kong lying back down on the print bed (this seemed to be the best orientation for ease of support removal). I would recommend printing with more than 3 perimeters (perhaps 5) so the print is strong enough to avoid damage during support removal. I printed with 3 perimeters and the tie and a fist broke off while I removed the supports despite my best efforts to be gentle.

I've uploaded the .3mf file of my print bed setup that can be loaded as a project in PrussaSlicer. This has 4.2mm diameter holes in the feet to attach DK to a stand (the model cannot easily stand on its own unless on an inclined surface, it tends to lean forward).

I've also uploaded the PrussaSlicer configuration I used (although this is also included in the .3mf file)

Post-Printing

Supports do take a while to remove on this (it took me about an hour) and care must be taken to avoid damage to the model. I accidentally broke off the tie and one of the fists while removing the supports, but I was able to easily glue them back on.

Painted with acrylic paints. Tried to give it a fur texture, but it wound up looking more like plaster. I like it though.
